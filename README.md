# Puppy Photo Gallery

A small static puppy gallery built from the PRD. Visitors can browse photos, open a larger detail view, and add JPG/PNG images with captions. Uploaded photos are stored in the browser with `localStorage`, so the site works without a backend.

## Features

- Responsive gallery: three columns on desktop, two on tablet, one on mobile
- JPG/PNG upload with a 5MB size limit
- Newest photos shown first
- Modal photo detail view with caption and upload timestamp
- About section with puppy personality details

## Run

Open `index.html` in a browser.
